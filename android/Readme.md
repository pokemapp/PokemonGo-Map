Installation
-----

 - Install the .apk on your device
 - Start it
 - Copy the token at the top of the screen
 - Start `example.py` with option `-f $token`.

For example:

```
python example.py -u $user -p $password -l "dam, amsterdam" -st 5 -f MYVERLONGTOKEN
```
